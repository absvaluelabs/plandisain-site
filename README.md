# plandisain-site

Static site for [PlanDisain](https://plandisain.fi) (kaavoitus, maankäytön digitalisaatio, vuorovaikutus).

Migration in progress from Weebly → static site on GitHub Pages, edited via [Claude Code](https://claude.com/claude-code).

## Layout

- `mirror/` — raw scrape of the current plandisain.fi site. Reference material for the rebuild. Not the deployed site.
- `index.html` — placeholder landing page during migration.
